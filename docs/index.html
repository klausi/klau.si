Hi! My name is Klausi. I work as Head of Technology Operations for jobiqo. I have a master's degree in software engineering from the Vienna University of Technology where I focused my soft skill part on Gender Studies. I'm interested in technology, politics and ethics.

You can contact me on Mastodon, Twitter or per email. My first name dot my last name (Klaus Purer) @protonmail.ch.

Some projects I work on:

  • Coder, a coding standards checker for Drupal
  • Mastodon Twitter Snyc, a tool to synchronize posts between Mastodon and Twitter
  • Rustnish, a Rust learning project with the example of a reverse proxy
  • GraphQL for Drupal, a Drupal module that implements a GraphQL server API
